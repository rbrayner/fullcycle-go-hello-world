## 1. Objective
Create a small docker image Go application, less than 2MB, printing a hello world.

## 2. The Docker Hub Repository

```shell
https://hub.docker.com/repository/docker/rbrayner/fullcycle-go-hello-world
```

## 3. Run

```shell
docker run --rm rbrayner/fullcycle-go-hello-world:latest
```

## 4. Build and Run Locally

To build and run this code locally, just type:

```shell
make all
```
